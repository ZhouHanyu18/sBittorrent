# sBittorrent

* 傻瓜式种子下载及资源搜索
* 基于libtorrent的简洁版BT下载工具
* 支持如下功能：
  * 种子资源搜索及一键下载
  * 文件名自动更换
  * 快捷定位文件路径
  * 生成.Torrent文件
  * 提取Magnet链接
